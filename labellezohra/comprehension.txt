La belle Zohra

Le but du jeu
Terminer les péchés capitaux => ben non, faut lire le readme.

Je clique sur une image de la table
=> afficahge d'un péché capital
=> je suis obligé de cliquer dessus
je clique dessus
=> affichage de l'image en grand
=> affichage du texte
on peut cliquer en haut ou en bas du texte pour le faire défiler
clic sur bouton droit pour sortir du texte
=> ça fait apparaître d'autres icônes

si je clique sur le bouton droit sur les icones
=> je reviens en arrière

si je clique sur le bouton droit sur les images de la table
=> il ne se passe rien du tout
