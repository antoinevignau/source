AppleSqueezer v2 SD Card Driver		  (c) 2026 by Brutal Deluxe__________________________________________________________________What can I do with the SD card port of my AppleSquezer v2 accelerator for the Apple IIgs?Perhaps one could set up four ProDOS partitions on a SD card and have them mounted into the Apple IIgs System Software.How to set up the SD Card____________________________________________1/ Upgrade your AppleSqueezer v2 card to Core v13.Get it from the Applesqueezer website. This version of the Core of the AppleSqueezer accelerator is mandatory to use if one wants to benefit from the best experience with the SD card port.Use the Control Panel AppleSqueezer to upgrade to Core v13. Then shutdown your computer.2/ Install the four-partition disk image onto your SD card.Download the four-partition disk image for your SD card on our website. Then use your preferred way to install the disk image onto your SD card. Let's consider your modern computer mounts it on /dev/disk4:	(sudo) diskutil umount /dev/disk4	(sudo) dd if=as4p32mb.bin of=/dev/disk4 bs=512Install the SD card into the port of the AppleSqueezer v2 accelerator card. Then, turn on your Apple IIgs computer.3/ Install the Apple IIgs GS/OS driver.Download the disk image with the installer from our website. Insert the disk and double-click the Installer program. Install the script. Reboot.You are now ready to use the SD card port of your AppleSqueezer v2 accelerator. FAQ________________________________________________________________Q/ Can I boot from the SD card?A/ Not yet. One step at a time.Q/ Can I mount disk images like other disk interfaces (CFFA, Booti, etc.)?A/ Not yet. One step at a time.Q/ What kind of partition model do you use?A/ This is a standard MS-DOS partition map. Note that the partition ID used for the ProDOS partitions is $20, a value that is reserved but not used.Q/ Can I use other partitions than a ProDOS one?A/ Yes, for sure. You have two options:	- the SD card has no partition map but a file system as from block 0,	- the SD card has up to four partitions with or without file systems on it.As the driver recognizes FAT12, FAT16, FAT32, ProDOS, and HFS partitions, you can do whichever combination with the card.Q/ Can I read, write, and format?A/ Yes, for sure.Q/ Is it a stable driver?A/ Yes, despite being still a beta release as there are some changes to apply to the partition recognition. For instance, if you have a partition map for four partitions and use only one, the Finder will tell you "4 - 1 active partition" times for image conflict. You can eject the other partitions.Q/ Is it a stable driver?A/ Yes, no SD card was hurt during writing and testing.Q/ Does it work with all SD card brands?A/ At least with two: Sandisk, and Nokia.Minimum system requirements_________________________________________The program requires at least 10KB of free memory.It has been tested on Apple IIgs ROM 01 & ROM 3 versions.It is compatible with System 6.x and above.Limited warranty___________________________________________________Many of the designations used by manufacturers and sellers to distinguish their products are claimed as trademarks.
